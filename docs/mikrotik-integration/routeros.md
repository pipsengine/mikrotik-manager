# MikroTik Integration

Supported connection methods:

- RouterOS REST API.
- RouterOS API.
- SSH.

Integration services:

- Discovery.
- Connection test.
- Inventory.
- Health.
- Configuration reader.
- Backup.
- Executor.
- Validator.
- Rollback.
- Command policy.

Execution is never immediate. The enforced order is read, analyze, backup, plan, approval, execute, validate, document and rollback when necessary.
