# API Reference

Primary modules:

- `auth` - JWT, refresh tokens, remember me, sessions, MFA readiness, failed-login and lockout model.
- `access-control` - roles, permissions and permission matrix.
- `mikrotik` - RouterOS REST API, API and SSH integration boundaries.
- `configuration` - RouterOS configuration domains.
- `ai-agent` - controlled AI orchestration.
- `change-management` - ITIL lifecycle and approval workflow.
- `backups` - automatic, manual, scheduled, encrypted and versioned backups.
- `monitoring` - CPU, memory, storage, temperature, interfaces, bandwidth, logs, alerts, health and availability.
- `security` - firewall audit, password audit, services audit, port exposure, weak configuration detection, compliance, drift and threat detection.
- `reports` - PDF, Excel and CSV reports.
- `audit` - immutable audit trail.
- `secrets` - encrypted secrets vault.
