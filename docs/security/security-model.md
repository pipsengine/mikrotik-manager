# Security Model

Security controls:

- Zero Trust posture.
- Permission-based RBAC for every page, button, API and action.
- Password policy, expiry, MFA readiness, account lockout and device tracking.
- HTTPS-only deployment, secure cookies, CSRF, CSP, rate limiting and input validation.
- Prisma and DTO validation for SQL injection protection.
- XSS protection and secure headers.
- Encrypted secrets vault.
- Immutable audit logs.
- Backup encryption.

Roles:

- Super Administrator
- Network Administrator
- Network Engineer
- Security Administrator
- Helpdesk
- Auditor
- Read Only User
- Approver
- Operations Manager
